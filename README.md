# simplex_with_static

для запуска приложения необходимо:
    сколнировать репозиторий в любую пустую папку
    в этой папке командой python3 -m venv venv создать вритуальную среду
    из этой папки активировать виртуальную среду командой source venv/bin/activate
    командой pip3 install -r requirements.txt установить все зависимотси
    выполнить команду export FLASK_APP=api.py
    командой flask run запустить flask сервер
